# Niprov
## Neurohackweek 2016
## Jasper J.F. van den Bosch


Download sample data
```shell
wget https://github.com/ilogue/niprov/archive/master.zip
unzip master.zip
cd niprov-master
```

Install Niprov
```shell
pip install niprov
```

Optional packages
```shell
pip install -r optional.txt # pydicom nibabel mne matplotlib
```
